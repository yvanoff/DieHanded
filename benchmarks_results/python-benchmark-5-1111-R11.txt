These results were generated by an old Python script I used before making the C++ run generator
The Python script had less features and notably no checks for BC level/RotG/Bad Seeds/Runes in possession of the player
For this reason I can't do Python benchmarks in any other conficguration than:
- 5bc
- all runes enabled
- both DLCs enabled
- challenge biomes set to random
However, this lack of options is also a blessing, because it means the results generated are quite trustworthy and approach the probability distribution of each
biome very well.
Compare to the file "results-5-1111-R11.txt". The results should be the same (both were generated using nb-iter=100000).
